# Lesson Exercises

## 09-01: Memory Usage

Python has several ways to store and manipulate data in array 
or dictionary format where items are located. 

Do a comparison of these common data types what kind of memory usage these have: 

1. `list`, 
2. `array`,  
3. `tuple` and
4. `dict`.

Create arrays which has at least 100 000 items but use only simple data types like `int` or `str`.

* Extra: Is there any better data type to use instead of these simple data types. 

## 09-02: Memory Consumption

Find out tools and libraries to use to check memory consumption in your program during runtime.
